# Code of Conduct

This project expects direct, respectful collaboration.

## Expected Behavior

* Be clear, constructive, and specific.
* Discuss technical disagreements through evidence and tradeoffs.
* Respect maintainers' time by keeping issues and pull requests focused.
* Avoid sharing private videos, credentials, service account files, or personal data in public issues.

## Unacceptable Behavior

* Harassment, threats, insults, or discriminatory language.
* Publishing someone else's private information.
* Posting secrets, copyrighted media, or credentials.
* Repeated off-topic or bad-faith comments.

Maintainers may edit, hide, or remove content and restrict participation when needed to keep the project useful and safe.
