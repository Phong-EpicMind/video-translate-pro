# FFmpeg Corresponding Source Offer

The standalone macOS builds of TranslateDub AI may bundle static FFmpeg and FFprobe binaries.

Bundled FFmpeg/FFprobe identification:

* Version string: `N-124530-gf435ce22e1-https://www.martin-riedl.de`
* FFmpeg commit: `f435ce22e1`
* Reported GPL flags: `--enable-gpl --enable-version3`
* Build script: [git.martin-riedl.de/ffmpeg/build-script](https://git.martin-riedl.de/ffmpeg/build-script)
* Upstream source: [git.ffmpeg.org/ffmpeg.git](https://git.ffmpeg.org/ffmpeg.git)

For any binary release that includes these FFmpeg/FFprobe executables, the release maintainer should provide the Corresponding Source for the exact FFmpeg commit and build configuration by one of these methods:

* Attach a source archive to the same GitHub Release, for example `ffmpeg-N-124530-gf435ce22e1-corresponding-source.tar.xz`.
* Provide a written source offer in the release notes, valid for at least three years, with a working contact path such as the repository issue tracker or maintainer email.

This offer applies only to the bundled FFmpeg/FFprobe binaries. TranslateDub AI's own source code is licensed separately under the MIT License.
